Clases y Objetos

1 / 15

¿Qué es una Clase?

Una plantilla o un plano para crear objetos. Define atributos (datos) y métodos (comportamientos).

2 / 15

¿Qué es un Objeto?

Una instancia de una clase. Es la "construcción" real creada a partir del "plano".

3 / 15

Definir una Clase: class

Se usa la palabra clave `class` seguida del nombre de la clase (por convención, en CamelCase).

class Perro:
    pass # 'pass' significa que no tiene contenido aún
4 / 15

Crear un Objeto (Instancia)

Se "llama" a la clase como si fuera una función para crear una nueva instancia.

class Perro:
    pass

mi_perro = Perro()
otro_perro = Perro()

print(type(mi_perro)) #
5 / 15

El Constructor: __init__

Un método especial que se ejecuta automáticamente al crear un objeto. Se usa para inicializar los atributos.

class Perro:
    def __init__(self, nombre, raza):
        self.nombre = nombre
        self.raza = raza
6 / 15

El Parámetro self

Representa la instancia del objeto en sí. Permite que cada objeto acceda a sus propios atributos y métodos.

# self.nombre se refiere al nombre del objeto específico
self.nombre = nombre
7 / 15

Atributos de Instancia

Son las variables que pertenecen a cada objeto individual (ej. `nombre`, `raza`).

fido = Perro("Fido", "Golden")
buddy = Perro("Buddy", "Labrador")

print(fido.nombre) # Fido
print(buddy.nombre) # Buddy
8 / 15

Métodos de Instancia

Son las funciones que pertenecen a la clase y que operan sobre los datos del objeto (sus atributos).

class Perro:
    def __init__(self, nombre):
        self.nombre = nombre
        
    def ladrar(self):
        print(f"{self.nombre} dice: ¡Guau!")
9 / 15

Llamar a un Método

Se accede a los métodos de un objeto usando la notación de punto.

fido = Perro("Fido")
fido.ladrar() # Fido dice: ¡Guau!
10 / 15

Herencia: Reutilizar Código

Permite que una clase (hija) herede los atributos y métodos de otra clase (padre).

11 / 15

Clase Padre (Superclase)

La clase de la que se hereda.

class Animal:
    def __init__(self, nombre):
        self.nombre = nombre
    
    def comer(self):
        print(f"{self.nombre} está comiendo.")
12 / 15

Clase Hija (Subclase)

La clase que hereda. Se especifica la clase padre entre paréntesis.

class Perro(Animal): # Perro hereda de Animal
    def ladrar(self):
        print("¡Guau!")
13 / 15

La Herencia en Acción

El objeto `Perro` ahora tiene acceso a los métodos de `Animal` y de `Perro`.

mi_perro = Perro("Fido")
mi_perro.comer()  # Método heredado de Animal
mi_perro.ladrar() # Método propio de Perro
14 / 15

Sobrescribir Métodos

Una clase hija puede redefinir un método de la clase padre para que se comporte de manera diferente.

class Gato(Animal):
    def comer(self): # Sobrescribimos el método comer
        print(f"{self.nombre} está comiendo atún.")

mi_gato = Gato("Misi")
mi_gato.comer()
